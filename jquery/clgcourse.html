PICT B.E. Course Structure

First Year (FE)

  • Engineering Mathematics-I

    Fundamentals of calculus, differential equations, and complex numbers.

  • Basic Electrical Engineering

    Basics of electrical circuits, transformers, and motors.

  • Engineering Physics

    Concepts of optics, quantum mechanics, and materials.

Second Year (SE)

  • Data Structures

    Introduction to arrays, linked lists, trees, and graphs.

  • Computer Organization

    Study of microprocessors, memory hierarchy, and instruction sets.

  • Database Management Systems

    Relational models, SQL, normalization, and transactions.

Third Year (TE)

  • Operating Systems

    Processes, threads, memory management, and file systems.

  • Computer Networks

    OSI model, TCP/IP, routing, and network security.

  • Web Technologies

    HTML, CSS, JavaScript, and server-side scripting.

Final Year (BE)

  • Machine Learning

    Supervised and unsupervised learning, regression, and classification.

  • Cloud Computing

    Virtualization, distributed systems, and cloud architecture.

  • Project Work

    Capstone project involving real-world application of learned skills.
